# Document Merging POC

Minimal React + Express POC for merging PDF and DOCX files.

Run backend:

```powershell
cd backend
npm install
npm start
```

Run frontend (in another terminal):

```powershell
cd frontend
npm install
npm run dev
```

Open the Vite app (usually http://localhost:5173), select merge type, choose files, click Merge, then Download.
